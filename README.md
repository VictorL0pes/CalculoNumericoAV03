# Cálculo Numérico - AV03 - Unidade 3

Simulação da suspensão de um carro (oscilador harmônico amortecido) como aplicação de EDOs à física de jogos.

EDO: `m*x'' + c*x' + k*x = 0`

Dois métodos numéricos foram implementados: 
- Euler explícito 
- Runge-Kutta de 4ª ordem (RK4)

## Como rodar

```
pip install matplotlib
python main.py
```

Os gráficos são gerados na pasta `graficos/`.
